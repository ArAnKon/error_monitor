Настройки Jira Integration

Настройте подключение к Jira для отправки ошибок из расширения

Пример: https://mycompany.atlassian.net
🔗 Как получить API токен?
Ключ вашего проекта в Jira (например: PROJ, TEST)
Выберите тип задачи: Баг или Улучшение
Account ID пользователя или email
Названия компонентов через запятую
Метки через запятую (без пробелов)
Пример: customfield_10000 для Story Points
